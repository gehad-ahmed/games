# 🎮 Free Games Explorer

> 🔗 **Live Demo:** [https://gehad-ahmed.github.io/games/](https://gehad-ahmed.github.io/games/)

A web app to browse and discover **free-to-play games**, with details for each title, built with vanilla JavaScript and the FreeToGame API.

## 📖 Overview

Browse a live catalog of free games fetched from the **[FreeToGame API](https://www.freetogame.com/api-doc)**. Click any game to open a details page with its description, screenshots, publisher, release date and more. The data layer is organized using ES6 **classes**.

## ✨ Features

- 🕹️ Live grid of free-to-play games with cover art
- 📄 Dedicated game details page (description, genre, platform, publisher, release date)
- ⚡ Asynchronous data loading with Fetch API and `async/await`
- 🎨 Clean responsive UI

## 🛠️ Tech Stack

- HTML5 & CSS3
- Bootstrap
- Vanilla JavaScript (ES6+ classes, Fetch API)
- [FreeToGame API](https://www.freetogame.com/api-doc)

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/games.git
cd games
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).
